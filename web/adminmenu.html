About

The Voice Banking application is an online banking application with voice recognition which aims to facilitate the use
of the application in order to allow customers and specially customers with disabilities to break this limitation
caused by disability to manage their online bank account using only their voice.
The application uses OTP authentication for security purposes and provides a special invoice
to any customer or business to track every activity that will be performed using our voice banking application.
